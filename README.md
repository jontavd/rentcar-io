# rentcar-io

Personal Project to search for prices on GoldCar site using the UI Test Tool Node.JS + WebDriver.IO

## Installation

1. Clone the repository.

   ```sh
   git clone git@github.com:jontavd/rentcar-io.git

1. Install dependencies

   ```sh
   npm install

1. Run the tests.

   ```sh
   npm run wdio
